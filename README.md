# Supermarket Sales Analysis (Capstone Project)

Final project (IOC11) for the **Skills Bootcamp in Cloud Computing and Data Analytics** at Edge Hill University (Cohort 3, 2022).

## Project Overview
This project involved the end-to-end analysis of a supermarket sales dataset. I had to undertake data cleaning and normalisation before presenting actionable insights through statistical visualisation.

>NB: My actual code was written as a Python file but I have transposed it into a Jupyter notebook for display purposes. :)

## Technical Highlights
* **Data Engineering:** Uses end-to-end data normalisation and cleaning to ensure data integrity for analysis.
* **Statistical Analysis:** Leverages **Pandas** for data manipulation and **Matplotlib/Seaborn** for trend analysis and visual representation.
* **Business Intelligence:** Identifies key retail trends, which I presented in a timed 6-minute narrated presentation for a non-technical panel.

## Tech Stack
* **Python** (Core Logic) 
* **Pandas** (Data Normalisation) 
* **Matplotlib & Seaborn** (Visualisation) 
* **Jupyter Notebook** (Demonstration Environment)
